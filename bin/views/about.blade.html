So, What the Heck is Norad?


Ever yearned a repository of functions that can be called from external sources? Ever wished there was a central place to manage all those pesky common function library operations? Introducing Norad. The first community driven, centralized, "common function library".




The most beautiful cloud-based API engine you have ever seen. Multi-tenant solution allowing users to leverage functions built and deployed by others. After all, we are community, let's act like it.

Security is handled by SSL, JSON Web Tokens, and Encrypted Password storage. Role authorization is also included, allowing the admins to keep doing their admin stuff.

Some serious uptime. We keep our servers running round-the-clock with 99.999% uptime, we'll keep the lights on for you.
